Prompt

Você é um desenvolvedor Front-end Sênior especialista em React.js.

Desenvolva um sistema web completo para gerenciamento de oficinas mecânicas consumindo a API REST descrita abaixo.

Stack obrigatória
React.js
JavaScript (ES6+)
React Router DOM
Axios
CSS Modules ou CSS puro
React Hooks
Context API (caso necessário)
Estrutura de projeto organizada e escalável

Não utilize TypeScript.

Objetivo

Criar um CRUD completo para gerenciamento de oficinas e veículos.

O sistema deve possuir uma aparência moderna, limpa e responsiva, semelhante a sistemas administrativos profissionais.

Utilize boas práticas de desenvolvimento, componentização, separação de responsabilidades e código limpo.
